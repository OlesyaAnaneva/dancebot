# Telegram Dance School Bot 🩰

Бот для записи на занятия в школу танцев Анны.

## 📋 Функциональность

### Для учеников:
- 👤 Регистрация и выбор уровня подготовки
- 📅 Просмотр доступных занятий (фильтрация по уровню)
- ✅ Запись на групповые, индивидуальные занятия, интенсивы
- 📋 Просмотр своих записей
- 📍 Информация о студии, что взять с собой, контакты

### Для администратора (Ани):
- 👑 Панель управления через Telegram
- 📋 Подтверждение оплат
- ➕ Создание и управление активностями
- 📊 Статистика и отчеты
- 📢 Рассылка уведомлений
